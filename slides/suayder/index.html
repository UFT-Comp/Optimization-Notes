Biblioteca web para otimização e aprendizado

Objetivo do estágio:

Desenvolver uma plataforma web com algoritmos de otimização e de aprendizado de máquina, oferecendo ao usuário uma iterface para testes de algoritmos e com capitulos para fundamentação teórica sobre os algoritmos

Tecnologias de desenvolvimento


  • Desenvolvimento de algoritmos em:
    • C
    • C++
    • Python - SciKit
  • Markdowm
  • Pandoc

Resultados:

O que foi desenvovido
  • Um site hospedado no: Github onde o mesmo inclue:
    • Um livro com uma série de capitulos os quais envolvem conteúdos como:
      • Fundamentação matemática
      • Fundamentos de otimização
      • Gradiente descendente
      • Regressão linear
      • Differential Evolution
  • Códigos de exemplos implementados sobre cada tópico
