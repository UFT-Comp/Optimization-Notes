Plataforma online para aprendizado sobre otimização

Objetivos do projeto


  • Desenvolvimento de um ambiente online para aprendizado sobre otimização
  • Estudo e desenvolvimento de diversos algoritmos de otimização, com foco em performance e modularidade
  • Desenvolvimento de uma biblioteca especializada em otimização contínua (nlpp)

nlpp


  • A linguagem de programação utilizada foi o C++, especificamente o C++ 11/14
  • A biblioteca foi desenvolvida para conter somente templates em headers
    • Isto é, uma header only library
  • Usando a biblioteca de álgebra linear Eigen, a nlpp é bem rápida!

nlpp


  • Diversos algoritmos de otimização foram desenvolvidos em forma de templates
    • Gradient Descent
    • Newton's method
    • Conjugate Gradient
    • BFGS
    • L-BFGS

Interface web


  • A interface web foi construída para ser acessada na forma de livro, com alguns exemplos online
  • As ferramentas usadas foram:
    • Vue
    • Emscripten
    • Markdown
    • Chart.js
    • Reveal.js

Interface web


  • O site é um gitpage
  • Isso é, um site estático
  • Sem servidor, sem banco de dados
  • HTML, CSS e JS

Comunicação C++/JS


  • A ferramenta Emscriten foi utilizada para compilar o código desenvolvido em C++ para Java Script
  • Dessa forma, podemos acessar classes e algoritmos implementados em C++ diretamente de JS
